Struktur data merupakan cara programmer untuk mempresentasikan suatu data, dan memudahkan programmer untuk menyelesaikan permaslahan dengan menggunakan struktur data tersebut.

Struktur data Linear Structure merupakan struktur data sederhana dimana ketika data ditambahkan maka data tersebut berada pada posisi relatif terhadap data yang lain dari struktur tersebut. Linear structure ini memiliki dua buah ujung. Dua buah ujung ini memiliki beberapa istilah seperti left-right, front-rear, atsupun top-bottom.
