Rekursi

Rekursi terjadi ketika sebuah fungsi memanggil dirinya sendiri.

Rekursi adalah konsep umum dalam matematika dan pemrograman karena memungkinkan kita melakukan perulangan/iterasi melalui data hingga mencapai suatu hasil.

Catatan: rekursi harus ditulis hati-hati agar tidak berujung fungsi tak berakhir (infinite recursion) atau menyebabkan stack overflow.

Base Case & Step Rekursif

  • Base Case: kondisi yang menghentikan rekursi.
  • Step Rekursif: memanggil fungsi dirinya sendiri dengan argumen yang dimodifikasi.

NB: pastikan base case selalu pada akhirnya terpenuhi.

def hanoi(n, asal, bantu, tujuan):

    # Base Case

    if n == 1:

        print(f"Pindah disk 1 dari {asal} ke {tujuan}")

    # Step Rekursif

    else:

        hanoi(n-1, asal, tujuan, bantu)

        print(f"Pindah disk {n} dari {asal} ke {tujuan}")

        hanoi(n-1, bantu, asal, tujuan)



hanoi(3, 'A', 'B', 'C')

Contoh Rekursi: Faktorial & Fibonacci (sesuai notebook)

def fak(n):

    # Base Case

    if n == 1:

        return 1

    elif n%2== 0:

        return fak(n-1)

    # Step Rekursif

    else:

        return n*fak(n-1)



print(fak(8)) # 7*5*3*1



def factorial(n):

    # Base case

    if n == 0 or n == 1:

        return 1

    # Step Rekursif

    else:

        return n * factorial(n - 1)



print(factorial(5))

Rekursi dengan Daftar

Rekursi bisa memproses daftar dengan menangani satu elemen pada satu waktu.

# Hitung jumlah semua elemen dalam sebuah daftar:

def sum_list(numbers):

    if len(numbers) == 0:

        return 0

    else:

        return numbers[0] + sum_list(numbers[1:])



my_list = [1, 2, 3, 4, 5]

print(sum_list(my_list))



# Temukan nilai maksimum dalam sebuah daftar:

def find_max(numbers):

    if len(numbers) == 1:

        return numbers[0]

    else:

        max_of_rest = find_max(numbers[1:])

    return numbers[0] if numbers[0] > max_of_rest else max_of_rest



my_list = [3, 7, 2, 9, 1]

print(find_max(my_list))

Batasan Rekursi

Python memiliki batasan kedalaman rekursi default (sekitar 1000 panggilan rekursif), bisa berbeda tergantung environment.

Jika perlu lebih dalam, bisa menaikkan batasnya, tetapi hati-hati karena bisa menyebabkan RAM penuh, crash, atau kernel mati karena rekursi menyimpan pemanggilan fungsi di memori.

import sys

print(sys.getrecursionlimit())



import sys

sys.setrecursionlimit(2000)

print(sys.getrecursionlimit())

# eksponen

def eksponen(x, n):

    if n == 0:

        return 1

    else:

        return x * eksponen(x, n - 1)



print(eksponen(2, 3))



# Faktorial

def faktorial(n):

    if n == 1 or n == 0:

        return 1

    else:

        return n * faktorial(n - 1)



print(faktorial(5))



# fibonacci

def fibonacci(n):

    if n == 0:

        return 0

    elif n == 1:

        return 1

    else:

        return fibonacci(n-1) + fibonacci(n-2)



print(fibonacci(6))



# deret aritmatika

def deret_aritmatika(a, b, n):

    if n == 1:

        return a

    else:

        return deret_aritmatika(a, b, n-1) + b



print(deret_aritmatika(1, 4, 3))
