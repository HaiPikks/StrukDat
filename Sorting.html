Shorting

Sorting adalah proses mengurutkan data berdasarkan aturan tertentu.

Contoh aturan pengurutan:

  • Ascending (kecil → besar): 1, 2, 3, 4
  • Descending (besar → kecil): 4, 3, 2, 1

Bubble Sort

Bubble Sort adalah algoritma yang mengurutkan array dari nilai terendah ke nilai tertinggi (ascending) dengan membandingkan dua elemen bersebelahan, lalu menukar jika urutannya salah.

Cara kerjanya:

  1. Telusuri array satu per satu.
  2. Bandingkan setiap elemen dengan elemen berikutnya.
  3. Jika elemen saat ini lebih “salah urut” dibanding berikutnya, tukar.
  4. Ulangi sampai seluruh array terurut.
data = [5, 3, 8, 1]

for i in range(len(data)):

    for j in range(len(data)-1):

        if data[j] > data[j+1]:

            data[j], data[j+1] = data[j+1], data[j]

print(data)

Selection Sort

Selection Sort menemukan nilai terendah (atau terkecil) dalam sebuah array, lalu memindahkannya ke bagian depan array yang belum diurutkan.

Cara kerjanya:

  1. Telusuri array untuk menemukan nilai terendah.
  2. Pindahkan nilai terendah ke posisi depan dari bagian yang belum terurut.
  3. Ulangi sampai seluruh array terurut.
data = [5, 3, 8, 1]

for i in range(len(data)):

    min_index = i

    for j in range(i+1, len(data)):

        if data[j] < data[min_index]:

            min_index = j

    data[i], data[min_index] = data[min_index], data[i]

print(data)

Insertion Sort

Insertion Sort mengurutkan array dengan cara menyimpan bagian yang sudah terurut di satu sisi, lalu memasukkan elemen berikutnya ke posisi yang benar (mirip menyusun kartu).

Cara kerjanya:

  1. Ambil elemen berikutnya dari bagian yang belum terurut.
  2. Geser elemen terurut untuk memberi tempat.
  3. Tempatkan elemen pada posisi yang tepat.
data = [5, 3, 8, 1]

for i in range(1, len(data)):

    key = data[i]

    j = i - 1

    while j >= 0 and key < data[j]:

        data[j + 1] = data[j]

        j -= 1

    data[j + 1] = key

Quick Sort

Quicksort adalah salah satu algoritma pengurutan tercepat. Quicksort memilih elemen sebagai pivot, lalu memisahkan data menjadi bagian kiri (lebih kecil dari pivot) dan kanan (lebih besar dari pivot). Proses dilakukan rekursif.

Rekursi terjadi ketika sebuah fungsi memanggil dirinya sendiri.

Cara kerjanya:

  1. Pilih nilai sebagai pivot.
  2. Pisahkan elemen selain pivot: kiri (lebih kecil), kanan (lebih besar).
  3. Tukar/penempatan pivot agar berada di antara dua bagian.
  4. Lakukan rekursif pada sub-array kiri dan kanan.

Pengertian Pivot

Pivot adalah nilai pembanding pada Quick Sort untuk membagi data menjadi:

  • lebih kecil dari pivot
  • lebih besar dari pivot

Contoh: Data [5,0,1,10,7,2], pivot = 5 → [0,1,2] 5 [10,7]

def quick_sort(lst):

    if len(lst) <= 1:

        return lst



    pivot = lst[0]



    kiri = [x for x in lst[1:] if x < pivot]



    kanan = [x for x in lst[1:] if x >= pivot]



    return quick_sort(kiri) + [pivot] + quick_sort(kanan)

Merge Sort

Merge Sort adalah algoritma bagi-dan-taklukkan (divide & conquer): memecah array menjadi bagian kecil, lalu menggabungkannya kembali secara terurut.

Bagi: memecah sampai sub-array tinggal 1 elemen.

Conquer: menggabungkan dua bagian kecil menjadi urut.

Cara kerjanya:

  1. Bagilah array menjadi dua sub-array.
  2. Lanjutkan pembagian sampai masing-masing sub-array berukuran kecil.
  3. Gabungkan dua sub-array dengan menempatkan nilai lebih kecil lebih dulu.
  4. Ulangi sampai semua bagian tergabung.
def merge_sort(lst):

    if len(lst) <= 1:

        return lst



    mid = len(lst)//2



    kiri = merge_sort(lst[:mid])

    kanan = merge_sort(lst[mid:])



    hasil = []

    while kiri and kanan:

        if kiri[0] < kanan[0]:

            hasil.append(kiri.pop(0))

        else:

            hasil.append(kanan.pop(0))



    return hasil + kiri + kanan

Search

“search” dalam pemrograman itu artinya: mencari data.

Contoh:

  • mencari angka 7 dalam list
  • mencari nama dalam data
  • mencari posisi/index data

Linear Search

Linear search (sequential search) adalah algoritma pencarian paling sederhana: memeriksa setiap elemen satu per satu.

Cara kerjanya:

  1. Telusuri array dari awal.
  2. Bandingkan nilai elemen dengan nilai target.
  3. Jika ditemukan, kembalikan indeks.
  4. Jika tidak ditemukan sampai akhir, kembalikan -1.

Catatan: jika array sudah terurut, biasanya lebih cepat memakai Binary Search.

lst = [5,0,1,10,7,2]

cari = 7

for i in range(len(lst)):

    if lst[i] == cari:

        print("Data ditemukan di index", i)

Binary Search

Binary search mencari di array yang sudah diurutkan dan mengembalikan indeks target.

Binary search memeriksa nilai di tengah, lalu mempersempit area pencarian menjadi setengah dari sebelumnya.

Cara kerjanya:

  1. Periksa nilai tengah.
  2. Jika target lebih rendah → cari di kiri, jika lebih tinggi → cari di kanan.
  3. Ulangi sampai ditemukan atau area kosong.
  4. Jika tidak ditemukan → kembalikan -1.
lst = [1,2,3,4,5,6,7,8,9]

cari = 7

awal = 0

akhir = len(lst)-1

    tengah = (awal + akhir)//2

        print("Data ditemukan")

        break

    elif cari < lst[tengah]:

        akhir = tengah - 1

    else:

        awal = tengah + 1
def binarySearch(arr, targetVal):

    left = 0

    right = len(arr) - 1

    while left <= right:

        mid = (left + right) // 2

        if arr[mid] == targetVal:

            return mid

        if arr[mid] < targetVal:

            left = mid + 1

        else:

            right = mid - 1

    return -1

def bubblesort(p):

    n = len(p)

    for i in range(n):

        for j in range(n):

            k = j + 1

            while k < n and (p[k] % 2) != (p[j] % 2):

                k += 1

            if k < n:

                if p[j] % 2 == 0:

                    if p[j] > p[k]:

                        p[j], p[k] = p[k], p[j]

                else:

                    if p[j] < p[k]:

                        p[j], p[k] = p[k], p[j]

    return p

def insertionzigzag(p):

    n = len(p)

    for i in range(1, n):

        j = i

        while j > 0:

            if (j-1) % 2 == 0:

                if p[j-1] < p[j]:

                    p[j-1], p[j] = p[j], p[j-1]

                else:

                    break

            else:

                if p[j-1] > p[j]:

                    p[j-1], p[j] = p[j], p[j-1]

                else:

                    break

            j -= 1

    return p
