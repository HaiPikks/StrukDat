Queue

Queue adalah struktur data yang menggunakan prinsip FIFO (First In First Out): data yang masuk pertama akan keluar pertama.

Struktur Queue

  • Front → penunjuk elemen paling depan (yang akan keluar)
  • Rear → penunjuk elemen paling belakang (tempat masuk data)
  • Array/List → tempat menyimpan data

Operasi Dasar Queue

Enqueue

  • Menambahkan data ke rear

Contoh:

  • Queue kosong → []
  • Enqueue A → [A]
  • Enqueue B → [A, B]

Dequeue

  • Menghapus data dari front

Contoh:

  • [A, B, C] → Dequeue → [B, C]

Peek / Front

  • Melihat data paling depan tanpa menghapus

Contoh: [A, B, C] → Peek = A

IsEmpty

  • Cek apakah queue kosong

Algoritma Queue (Enqueue & Dequeue)

Enqueue (Masuk ke Rear)

  • Cek apakah queue penuh
  • Jika tidak penuh, tambahkan data ke rear
  • Pembaruan posisi rear

Dequeue (Keluar dari Front)

  • Cek apakah queue kosong
  • Jika tidak kosong, ambil data dari front
  • Pembaruan posisi front

Contoh Enqueue & Dequeue (Queue)

Gunakan operasi queue untuk memodelkan proses masuk (enqueue) dan keluar (dequeue).

from collections import deque


def hot_potato(daftar_nama, k):


    q_permainan = deque()

    # memasukkan pemain ke queue
    for nama in daftar_nama:
        q_permainan.append(nama)

    # permainan berlangsung sampai tersisa 1 pemain
    while len(q_permainan) > 1:

        # memindahkan pemain dari depan ke belakang
        for i in range(k - 1):
            pindah = q_permainan.popleft()
            q_permainan.append(pindah)

        # pemain tereliminasi
        tereliminasi = q_permainan.popleft()
        print(f"Pemain nomor {tereliminasi} keluar dari lingkaran.")

    # pemenang
    return q_permainan.popleft()


n = 5
k = 2

pemain = list(range(1, n + 1))

# pakai k, bukan n
pemenang = hot_potato(pemain, k)

print("------------------------------")
print("Pemenang adalah pemain nomor:", pemenang)

# Output:
# Pemain nomor 2 keluar dari lingkaran.
# Pemain nomor 4 keluar dari lingkaran.
# Pemain nomor 1 keluar dari lingkaran.
# Pemain nomor 5 keluar dari lingkaran.
# Pemenang adalah pemain nomor: 3

Jenis-Jenis Queue

  1. Linear Queue
    • Antrian biasa
    • Tidak bisa reuse space kosong
  2. Circular Queue
    • Bentuk melingkar
    • Bisa menggunakan kembali ruang kosong
  3. Priority Queue
    • Setiap data punya prioritas
    • Yang prioritas tinggi keluar dulu

Soal Latihan: Hot Potato

  • Sekelompok anak duduk melingkar, tiap peserta bernomor 1 sampai n.
  • Pemain nomor awal memegang kentang panas, lalu mengoper ke pemain berikutnya searah jarum jam.
  • Pada hitungan ke-k, pemain yang memegang kentang keluar.
  • Ulangi sampai tersisa 1 pemain (pemenang).

Output: nomor pemain yang menang.

from collections import deque


def hot_potato(daftar_nama, k):


    q_permainan = queue()

    for nama in daftar_nama:

        enqueue(q_permainan, nama)

    while len(q_permainan) > 1:

        for i in range(k - 1):

            pindah = dequeue(q_permainan)

            enqueue(q_permainan, pindah)

        tereliminasi = dequeue(q_permainan)

        print(f"Pemain nomor {tereliminasi} keluar dari lingkaran.")

    return dequeue(q_permainan)

n = 5

k = 2

pemain = list(range(1, n + 1))

pemenang = hot_potato(pemain, n)

print("------------------------------")

print("Pemenang adalah pemain nomor:", pemenang)
