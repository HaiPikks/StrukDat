Hashing

Hashing adalah teknik untuk menyimpan dan mengambil data secara efisien menggunakan fungsi hash agar akses cepat.

  • Pemetaan data ke indeks tabel hash (array) memakai fungsi hash.
  • Rata-rata operasi pencarian/penyisipan/penghapusan bisa mendekati O(1).
  • Dipakai untuk kumpulan item (key) dan kamus (key-value).

Contoh Modulo

Fungsi hash: H(x) = x % 10 mengubah bilangan besar menjadi nilai 0 sampai 9.

Ciri-ciri Hashing

  1. Output panjang tetap
  2. Sulit dibalik (resistant terhadap reverse)
  3. Perubahan kecil bisa menghasilkan output sangat berbeda (misal “halo” vs “Halo”)
  4. Deterministik: input sama → output sama

Contoh Penggunaan

  • Keamanan password (disimpan dalam bentuk hash)
  • Penyimpanan data/database (mempercepat pencarian)
  • Cek integritas file (agar file tidak berubah)

Algoritma hashing terkenal: MD5 (tidak aman), SHA-1 (mulai ditinggalkan), SHA-256 (lebih aman).

Analogi

Hashing seperti memasukkan dokumen ke mesin penghancur → keluar potongan unik. Kamu bisa mengenali hasilnya, tapi tidak bisa menyusun ulang dokumen aslinya.

Collision

Collision terjadi ketika dua data berbeda menghasilkan hash yang sama.

Collision pada Hash Table

Di hash table, collision sering terjadi saat dua data masuk ke indeks array yang sama.

Contoh: ukuran tabel = 10 → 21 % 10 = 1 dan 31 % 10 = 1 (keduanya mau ke index 1).

class Hash:

    def __init__(self):

        self.panjang = 11

        self.table = self.hashTable()



    def hashFunction(self, data):

        return data % self.panjang



    def hashTable(self):

        return [[None] for i in range(self.panjang)]



    def chaining(self, data):

        for i in range(len(data)):

            idx = self.hashFunction(data[i])

            if self.table[idx] == [None]:

                self.table[idx] = [data[i]]

            else:

                self.table[idx].append(data[i])



    def searchHash(self, target):

        idx = self.hashFunction(target)

        if self.table[idx] != [None]:

            for i in range(len(self.table[idx])):

                if self.table[idx][i] == target:

                    return ("data berada di slot ke-", idx,

                            "dan indeks ke-", i)



        return False

data = [54, 26, 93, 17, 77, 31, 44, 55, 20]

varHash.chaining(data)

print(varHash.table)

print(varHash.searchHash(54))

print(varHash.searchHash(20))

print(varHash.searchHash(55))

print(varHash.searchHash(100))

Chaining

Chaining menangani collision dengan menyimpan data yang hash-nya sama dalam satu “rantai” (list/linked list) pada index tersebut.

Kelebihan: mudah dibuat, bisa menyimpan banyak collision, tidak perlu cari slot kosong lain.

Kekurangan: jika collision terlalu banyak, pencarian jadi lebih lambat.

class Chaining:

    def __init__(self):

        self.panjang = 11

        self.table = [[None] for i in range(self.panjang)]



    def hashFunction(self, data):

        return data % self.panjang



    def chaining(self, data):

        for i in range(len(data)):

            idx = self.hashFunction(data[i])

            if self.table[idx] == [None]:

                self.table[idx] = [data[i]]

            else:

                self.table[idx].append(data[i])

print("Chaining:")

print(chain.table)

Open Addressing

Open addressing menangani collision dengan mencari slot kosong lain di dalam tabel.

Jenis yang ada: Linear Probing, Quadratic Probing, Double Hashing.

class HashTable:

    def __init__(self, ukuran):

        self.ukuran = ukuran

        self.tabel = [None] * ukuran



    def fungsiHash(self, angka):

        return angka % self.ukuran



    def insert(self, angka):

        index = self.fungsiHash(angka)

        # cek collision

        while self.tabel[index] is not None:

            print(f"Collision di index {index} untuk angka {angka}")

            # pindah ke slot berikutnya

            index = (index + 1) % self.ukuran

        self.tabel[index] = angka

        print(f"{angka} masuk ke index {index}")



    def tampil(self):

        print("\nIsi Hash Table:")

        for i in range(self.ukuran):

            print(i, ":", self.tabel[i])



# Membuat objek hash table

hashTable = HashTable(10)

# Insert data

hashTable.insert(15)

hashTable.insert(25)

hashTable.insert(35)

# Tampilkan isi tabel

hashTable.tampil()

Close Address (Closed Addressing)

Closed Addressing pada dasarnya adalah chaining: data collision tetap berada di index yang sama.

class HashTable:

    def __init__(self, ukuran):

        self.ukuran = ukuran

        # setiap index berisi list kosong

        self.tabel = [[] for _ in range(ukuran)]



    def fungsiHash(self, angka):

        return angka % self.ukuran



    def insert(self, angka):

        index = self.fungsiHash(angka)

        # closed addressing / chaining

        self.tabel[index].append(angka)

        print(f"{angka} masuk ke index {index}")



    def tampil(self):

        print("\nIsi Hash Table:")

        for i in range(self.ukuran):

            print(i, ":", self.tabel[i])



# Membuat objek hash table

# Insert data

hashTable.insert(25)

hashTable.insert(35)

# Tampilkan isi tabel

hashTable.tampil()

Mid Square Method

Mid Square Method menentukan index dengan mengkuadratkan angka lalu mengambil digit tengah.

class HashTable:

    def __init__(self, ukuran):

        self.ukuran = ukuran

        self.tabel = [None] * ukuran



    def midSquareHash(self, angka):

        kuadrat = angka ** 2

        kuadratStr = str(kuadrat)

        # ambil 2 digit tengah

        tengah = len(kuadratStr) // 2

        digitTengah = kuadratStr[tengah-1:tengah+1]

        # jadikan index

        index = int(digitTengah) % self.ukuran

        return index



    def insert(self, angka):

        index = self.midSquareHash(angka)

        self.tabel[index] = angka

        print(f"{angka} masuk ke index {index}")



    def tampil(self):

        print("\nIsi Hash Table:")

        for i, data in enumerate(self.tabel):

            print(i, ":", data)



hashTable = HashTable(10)

hashTable.insert(44)

hashTable.insert(25)

hashTable.insert(12)

hashTable.tampil()

class Anagram:

    def cekAnagram(self, kata1, kata2):

        if len(kata1) != len(kata2):

            return False

        hashHuruf = {}

        for huruf in kata1:

            if huruf in hashHuruf:

                hashHuruf[huruf] += 1

            else:

                hashHuruf[huruf] = 1

        for huruf in kata2:

            if huruf not in hashHuruf:

                return False

            hashHuruf[huruf] -= 1

        return True



cek = Anagram()

print("\nDeteksi Anagram:")

print(cek.cekAnagram("listen", "silent"))

print(cek.cekAnagram("hello", "world"))
class Duplikat:

    def cekDuplikat(self, data):

        if len(data) != len(set(data)):

            return True

        return False



dup = Duplikat()

print("\nDeteksi Duplikat:")

print(dup.cekDuplikat([1, 2, 3, 4, 5]))

print(dup.cekDuplikat([1, 2, 3, 2, 5]))

print(dup.cekDuplikat(["a", "b", "a"]))
