Infix, Prefix, dan Postfix

Dalam stack, terdapat tiga bentuk penulisan ekspresi matematika: Infix, Prefix, dan Postfix untuk mempermudah proses komputasi.

1. Infix

Infix adalah bentuk ekspresi di mana operator berada di antara operand (misal: A + B * C). Umum dipakai dalam matematika sehari-hari.

Contoh dengan kurung untuk mengatur urutan: (A + B) * C.

Karena infix memakai prioritas operator, komputer sering mengubahnya ke prefix atau postfix agar mudah diproses.

def prioritas(operator):

    if operator == '+' or operator == '-':

        return 1

    elif operator == '*' or operator == '/':

        return 2

    return 0



def infixKePostfix(ekspresi):

    stack = []

    hasil = ""

    for char in ekspresi:

        # jika huruf atau angka

        if char.isalnum():

            hasil += char

        # jika kurung buka

        elif char == '(':

            stack.append(char)

        # jika kurung tutup

        elif char == ')':

            while stack and stack[-1] != '(':

                hasil += stack.pop()

            stack.pop()

        # jika operator

        else:

            while stack and prioritas(char) <= prioritas(stack[-1]):

                hasil += stack.pop()

            stack.append(char)

    # ambil sisa operator

    while stack:

        hasil += stack.pop()

    return hasil



infix = "A+B*C"

postfix = infixKePostfix(infix)

print("Infix  :", infix)

print("Postfix:", postfix)

2. Prefix

Prefix adalah bentuk ekspresi dengan operator ditulis sebelum operand. Contoh: Infix: A + B → Prefix: + A B.

Prefix tidak membutuhkan tanda kurung karena urutan operasi sudah jelas.

class Stack:

    def __init__(self):

        self.data = []



    def push(self, item):

        self.data.append(item)



    def pop(self):

        return self.data.pop()



    def peek(self):

        return self.data[-1]



    def kosong(self):

        return len(self.data) == 0



def prioritas(operator):

    if operator == '+' or operator == '-':

        return 1

    elif operator == '*' or operator == '/':

        return 2

    return 0



def infixKePostfix(ekspresi):

    stack = Stack()

    hasil = ""

    for char in ekspresi:

        # jika huruf/angka

        if char.isalnum():

            hasil += char

        # jika kurung buka

        elif char == '(':

            stack.push(char)

        # jika kurung tutup

        elif char == ')':

            while not stack.kosong() and stack.peek() != '(':

                hasil += stack.pop()

            stack.pop()

        # jika operator

        else:

            while (not stack.kosong() and

                prioritas(char) <= prioritas(stack.peek())):

                hasil += stack.pop()

            stack.push(char)

    # ambil sisa operator

    while not stack.kosong():

        hasil += stack.pop()

    return hasil



# Program utama

infix = "A+B*C"

postfix = infixKePostfix(infix)

print("Infix  :", infix)

print("Postfix:", postfix)

Konversi Infix → Prefix (fungsi infix_to_prefix)

# Fungsi prioritas operator

def prec(c):

    if c == '+' or c == '-':

        return 1

    elif c == '*' or c == '/':

        return 2

    else:

        return 0



# Fungsi infix ke prefix

def infix_to_prefix(infix):

    # 1. reverse

    infix = infix[::-1]

    # 2. tukar kurung

    infix = infix.replace('(', 'temp')

    infix = infix.replace(')', '(')

    infix = infix.replace('temp', ')')

    stack = []

    output = ""

    for char in infix:

        if char.isalnum():

            output += char

        elif char == '(':

            stack.append(char)

        elif char == ')':

            while stack and stack[-1] != '(':

                output += stack.pop()

            stack.pop()

        else:

            while stack and prec(stack[-1]) >= prec(char):

                output += stack.pop()

            stack.append(char)

    while stack:

        output += stack.pop()

    # 4. reverse lagi

    return output[::-1]



# Contoh

infix = "A+B*C"

print("Prefix:", infix_to_prefix(infix))

3. Postfix

Postfix adalah bentuk ekspresi di mana operator ditulis setelah operand. Contoh: Infix: A + B * C → Postfix: A B C * +.

infix = "A+B*C"

stack = []

output = ""

for char in infix:

    # jika huruf / angka → langsung ke output

    if char.isalnum():

        output += char

    # jika operator

    else:

        # kalau stack kosong atau prioritas lebih tinggi → push

        if not stack or char == '*' or char == '/':

            stack.append(char)

        else:

            # kalau prioritas lebih rendah → pop dulu

            while stack:

                output += stack.pop()

            stack.append(char)

# keluarkan sisa stack

while stack:

    output += stack.pop()

print("Postfix:", output)

Karena stack memiliki prinsip LIFO, cocok untuk menyimpan operator sementara.

Algoritma Konversi Infix → Postfix

  1. Buat stack kosong untuk operator.
  2. Baca ekspresi infix dari kiri ke kanan.
  3. Operand → langsung ke output.
  4. Operator → pop jika prioritas lebih tinggi/sama, lalu push.
  5. “(” → push.
  6. “)” → pop sampai “(”.
  7. Setelah selesai, pop semua operator tersisa ke output.
